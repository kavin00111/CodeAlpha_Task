# Python Mini Projects

A collection of beginner-friendly Python projects created to practice core programming concepts and problem-solving skills.

## Projects Included

### 1. Hangman Game

A simple text-based Hangman game where the player guesses a hidden word one letter at a time. The game selects a random word from a predefined list and gives the player a limited number of incorrect guesses.

**Concepts Used:**

* Random module
* While loops
* If-else statements
* Strings
* Lists
* User input/output

---

### 2. Stock Portfolio Tracker

A basic stock portfolio tracker that calculates the total investment value based on predefined stock prices and user-entered quantities. The result can also be saved to a text file.

**Concepts Used:**

* Dictionaries
* Input/Output
* Arithmetic operations
* File handling

---

### 3. Email Extractor

A Python script that extracts all email addresses from a text file using regular expressions and saves them to a separate file.

**Concepts Used:**

* Regular Expressions (`re`)
* File handling
* String processing

---

## Technologies Used

* Python 3
* Built-in Python Modules:

  * random
  * re

---
